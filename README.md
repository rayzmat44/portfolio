# Ray's Portfolio
## Overview

Welcome to my personal portfolio website! This project serves as a professional showcase of my web development skills, projects, and personal information. It is designed to highlight my abilities in HTML, CSS, and responsive design, providing a clear and organized interface for potential clients, collaborators, and employers.

The portfolio demonstrates my approach to clean, semantic code, professional project structure, and accessibility-conscious design.

## Live Demo

You can view the live version of my portfolio here:
https://raymondrayz719.netlify.app

# Features

## Responsive Design:
Fully responsive layout that adapts to desktop, tablet, and mobile devices using CSS Flexbox, Grid, and media queries.

## Semantic HTML:
Utilizes proper HTML5 elements (header, section, footer, nav) for accessibility and maintainability.

## Clean and Organized CSS:
Separate CSS file with consistent formatting, readable class names, and modular structure for easy updates.

## Live Preview:
Integrated with Live Server for real-time changes during development.

## Project Section:
Highlights personal or professional projects with live links, providing a snapshot of skills in action.

## Contact Section:
Provides easy access to email, GitHub profile, and other professional links.

Getting Started (For Developers)

To run this project locally:

Clone the repository:

git clone https://github.com/rayzmat44/portfolio.git

Navigate into the project folder:

cd portfolio

Open index.html in your browser, or use Live Server in VSCode for real-time updates:

Right-click index.html → Open with Live Server

Edit HTML, CSS, or JS as needed. Changes will be immediately visible in the browser.
