Selected Work

Project: School Management System

Context:

A mid-sized secondary school managing grading and fee records manually using spreadsheets.

The Problem

  • Time consuming report generation
  • Frequent calculation errors
  • Poor fee tracking visibility
  • Lack of role-based access

The Solution

Developed a centralized system that:

  • Allows teachers to input markd securely
  • Automatically calculates term annual grades
  • Generates printable report cards
  • Tracks fee Payments per student
  • Restricts access based on role

Technical Stack

  • React Frontend
  • Backend API
  • Database storage
  • Role-based authentication
  • PDF report generation

Outcome

  1. Reduced report preparation time significantly
  2. Improved data accuracy
  3. Increased administrative efficiency
  4. Created structured digital record keeping.
